Class Halls Account Reporter

A world of Warcraft Addon

Description

Class halls Account Reporter is a little project for World of Warcraft, it is written in Lua and uses some of the custom function provided by blizzard. The aim for this addon is to collect and store the data from all the characters of a player. By doing so, the player can then access this data from any character and view the status they are on.
