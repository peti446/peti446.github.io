Switch Switch

A world of Warcraft Addon

Description

This addon allows you to create multiple profiles of talents to be able to rapidly switch between them with a couple of clicks or commands (macros). Users will also have the ability to configure a prompt to ask them if they want to change when they have the wrong talents in a specific instance.

  • Create multiple profiles and store the talent build.
  • When switching talents and you are not in a rest zone, if tomes are available, a prompt may ask you to automatically use the tome to change talents.
  • Auto-prompt to change talents in instances.
  • Commands to change talents (for macro lovers)
